# Twitter Clone

## ***Technologies used:***

* **Java 8**
* **Spring Boot 2**
* **Freemaker**
* **SpringSecurity**
* **PostgreSQL**
* **Hibernate**
* **Bootstrap**
* **Bean validation, password encryption**
* **reCaptcha, rest client, rememberMe**
* **Integration Testing**
* **Pagination**
* **Module Testing**
* **HQL (JPQL) "Likes"**
